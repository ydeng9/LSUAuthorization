Key Request Procedure
<p>Key Request:</p><ol><li>Fill out the key request form.</li><li>All key request forms should be attached to an email and sent to the Auxiliary Services work order email, asfacilities@lsu.edu. Keys may not be ordered by telephone.</li><li>There is a charge for each key made. Please include an account number to be charged $5.00 on the key request form.</li><li>The Facilities Manager will approve and enter the request through the Facility Services work order tracking system.</li></ol><p>Key Pickup:</p><ol><li>Facility Services will notify the Facilities Manager when the keys are ready to be picked up.</li><li>The Facilities Manager will contact the individual/employee notifying them that their key is ready to be picked up at the Facility Services Building, Room 120.</li></ol>
